Introducción del OVI

El presente OVI permitirá reconocer conceptos importantes del curso de diseños de sitios web y se espera que al finalizar el material de apoyo que encontrara de defina el objetivo de aprendizaje que define el curso, y además se comprendan sus conceptos más importantes para el proceso académico, enfocando temas tales como GIT, GITHUB, HTML5 Y CSS3, con el fin de potenciar habilidades que corresponden al desarrollo de herramientas especializadas el línea.

Introducción diseños de sitios web

Es importante que tengamos en cuenta que el curso nos ofrece una iniciación básica en el campo del diseño web utilizando sistemas de control de versiones como GIT y la comunidad en línea GITHUB con su cliente para escritorio. Además se utilizaran los lenguajes de diseño web HTML5 y CSS3. El Diseño de sitios web es aquella actividad que consiste en planificar, diseñar, mantener y crear páginas web; y sirve para ofrecer a los usuarios una experiencia adecuada, suave y atractiva a la hora de moverse por una página en Internet.

OBJETIVOS

Objetivo general:

Crear un OVI del curso diseños de sitios web mediante formato HTML5 y CSS3, para ser visualizados en diferentes navegadores teniendo en cuenta su funcionabilidad y usabilidad.

Objetivos específicos:

* Plasmar conceptos claves para la comprensión del curso, tales como profundización e información complementaria sobre el curso de deseños de sitios web.
* Describir el uso de GIT y GITHUB para gestionar proyectos colaborativos de diseño web, HTML5 y CSS3 para creación de contenidos en formato web (Paginas y Sitios Web).
* Crear un sitio web donde se gestione la información ya antes mencionada con el fin de facilitar la comprensión.
